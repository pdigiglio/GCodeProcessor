

# GCode Processor [![Build Status](https://travis-ci.org/pdigiglio/GCodeProcessor.svg?branch=master)](https://travis-ci.org/pdigiglio/GCodeProcessor) [![Coverity Scan Build Status](https://scan.coverity.com/projects/10517/badge.svg)](https://scan.coverity.com/projects/pdigiglio-gcodeprocessor)

To compile it (on Linux):
```
$ mkdir build && cd build
$ cmake ..
$ make
```
This will create an executable file in `build/src` called `GCodeProcessor`.

Check out the documentation [here](https://pdigiglio.github.io/GCodeProcessor/index.html)!
